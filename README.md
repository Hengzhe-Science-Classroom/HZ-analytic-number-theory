# Analytic Number Theory · Interactive

**Prime number theorem, L-functions, and the Riemann zeta function**

**Status:** Planned

Part of [Hengzhe's Science Classroom](https://hengzhe-science-classroom.github.io/hub/) · Tier 5 · Research Frontier

## What You'll Learn

- Arithmetic functions and Dirichlet series
- The Riemann zeta function
- Distribution of primes
- The prime number theorem
- Dirichlet's theorem on primes in arithmetic progressions
- Sieve methods

## References

- Tom M. Apostol, *Introduction to Analytic Number Theory* (UTM, Springer)
- Harold Davenport, *Multiplicative Number Theory* (3rd ed., GTM 74, Springer)
- Iwaniec & Kowalski, *Analytic Number Theory* (AMS Colloquium Publications)
